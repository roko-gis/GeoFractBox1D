# GeoFractBox1D-
GeoFractBox1D is an open-source QGIS Processing tool for estimating one-dimensional fractal dimension of 1D spatial data and time-series using the classical box-counting method. It produces reproducible metrics including fractal dimension (D), scaling range, regression statistics, and diagnostic plots.
